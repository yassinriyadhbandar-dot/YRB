Flagship Project Award Winning

Graduation Project

Pilot Project for Using Fog & Cloud Computing in Drilling Operations

Overview

A data-driven pilot project focused on enhancing drilling performance by predicting the Rate of Penetration (ROP) using Machine Learning techniques. This innovative approach combines advanced computing architectures with petroleum engineering to optimize drilling operations.

Key Technologies & Tools

MATLAB Machine Learning Fog Computing Cloud Computing Web Dashboard Spotfire Software

Technical Implementation

  • Developed ML model in MATLAB to predict ROP based on real drilling parameters
  • Compared predicted ROP with actual field ROP to evaluate accuracy
  • Implemented Fog Computing for real-time, low-latency data processing
  • Utilized Cloud Computing for remote monitoring and visualization

Project Presentation Video

Impact & Value

  • • Supports future drilling decisions leveraging historical data
  • • Enhances drilling efficiency significantly
  • • Reduces non-productive time (NPT)
  • • Improves operational planning and decision-making
