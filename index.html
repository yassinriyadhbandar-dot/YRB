PROSPER

Well Performance & Nodal Analysis Software

PROSPER is a well performance and nodal analysis software developed by Petroleum Experts (Petex). It is used for designing, analyzing, and optimizing the performance of oil and gas wells.

Key Characteristics

  • Nodal Analysis - Inflow & outflow performance analysis
  • Flexible Well Modeling - Gas lift, ESP, rod pump
  • Multi-phase flow modeling (oil, gas, water)
  • Production optimization tools

Main Uses

Well Performance Artificial Lift Design Production Optimization Well Intervention
×

GAP

Surface Network Modeling Software

GAP is a surface network modeling software developed by Petroleum Experts (Petex). It models, analyzes, and optimizes the production network of oil and gas fields from reservoir to export.

Key Characteristics

  • Surface Network Modeling - Flowlines, pipelines, manifolds
  • Integration with PROSPER & MBAL
  • Multiphase Flow Simulation
  • Field-Wide Optimization

Main Uses

Network Analysis Production Optimization Field Development Troubleshooting
×

MBAL

Reservoir Engineering & Material Balance Software

MBAL is a reservoir engineering software by Petroleum Experts (Petex) for material balance analysis. It helps understand reservoir drive mechanisms, estimate hydrocarbons in place, and predict future performance.

Key Characteristics

  • Material Balance Calculations
  • Dynamic Reservoir Modeling
  • Hydrocarbon Volumes Estimation
  • Drive Mechanism Identification

Main Uses

Reservoir Characterization Production Forecasting History Matching Field Development
